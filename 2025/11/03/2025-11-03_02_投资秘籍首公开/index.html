正文：

今天这一章，我们给大家讲的是我自己投资特斯拉比特币、唯品会的具体实例。

讲投资特斯拉比特币和唯品会之前，我想先跟大家分享几个宏观经济形势。自从全球在上世纪七八十年代，美国主导的布林顿森林体系崩溃之后，全球的金融体系就已经与金本位脱钩了。所以全球的金融一直以来就是温和通胀的一个基本发展态势。所以整体即便以美元体系来看，美元也是每年都是超发的。像包括日本和欧洲已经常年保持低利率甚至负利率了，也就是你如果钱敢存钱，银行不仅每年不会给你利息，反而还会亏钱。而且同时政府还反复印钞让钱贬值。

所以，全世界最大的现象和问题，都是一个，就是钱太多。

现在先给大家讲讲我在美国当时投资的三个标的，特斯拉、比特币和唯品会。这三笔投资大家可以看到都是我在美国读书时候完成、做的几个case。我在美国期间是 2011 年到 2014 年，这大概三年时间完成的投资。可以很明显看到这些都是以美元计价的投资物品。

这里当然可以给大家说一下，即便大家在大陆，你住在大陆其实也是可以投资美元标的，包括美股，包括比特币都是可以投资的。

当然现在比特币在中国有很多交易所了，而同时美股其实你也可以使用，就是很多美股的证券公司，我在这也不做推荐了，就你找遍地都是。而且中国人在美国投资还有个好处是不用交税，因为我非美国居民是免税的，所以也可以投资美股。其中唯品会是中概股，特斯拉是一个美股，然后比特币是一种以美元为货币计价的一种密码学货币。

做投资入门一个一定要知道的，就是投资最重要的还是价格。一个人好不好，一个公司好不好很重要，那但是价格更重要。因为世界上其实投资标的也不是稀缺的，所以进入的价格非常重要。

我们来分析特斯拉、唯品会、比特币，其实都是好公司和好机会。分析好公司是一个非常复杂的话题。我们还是挑一个比较容易的来讲，因为分析好公司甚至也都是我自己要反复修炼的，这是无论你自己作为投资人、创业者，都要反复去修炼的一个东西。

这一节的时间很短，所以我们主要还是讲一讲购买的时机，就投资进入的时机。以三个例子来说明。

唯品会，当时购买的最好时机，也是我购买的时机了，是当时唯品会 2012 年年中流血上市的时候，这时候当时夸张到什么地步？当时唯品会主要的投资人红杉和 DCM 亏损都非常严重。之前是上市前的投资人，当时股价是 6.5 美元一股，而当时 b 轮的价格，都已经到了 10 美金一股了，所以损失非常惨重啊。但是大家可能会问，那唯品会的价格为什么会这么低？是不是这公司快垮了，是吧？你刚抄底进去就垮了，是吧？

这时候原因很复杂，但是其实简单说来就几个原因。第一个是中国公司整体在美国信誉出了问题，当时华尔街是非常看淡中概股的。当时也不光是中国公司，唯品会一家出现这些问题。

其次是中国电商集体不行，就包括京东访客整体都是下挫的，所以当时电商这个本身很危险。再三，唯品会本身自己处于 c 轮，融资不顺利，就唯品会并不是 c 轮融到一轮钱之后去资本市场上市了，而是 c 轮没融到钱，出此下策才去资本市场上市了。那你想这一轮的打击会是多大了？所以这时候唯品会的价格显然就会变得非常低了。

而这时候也为我们这种投资散户提供了机会，因为以前唯品会在一级市场的时候，你是没有办法购买的，而且像唯品会这样的公司也不可能接受散户的钱，尤其我一个当时在美国读书的学生，所以这时候买入物品会是最好的。

第二个例子，就是特斯拉买入的最好时机，其实就是 2012 年底到 2013 年初，当时也是个低谷，其实跟唯品会的情况差不多。第一个是电动车整个行业的低谷，当时大家可能也听过，当时奔驰、宝马很多这些，包括日本的丰田很多这些公司其实都在做电动车，做的都不顺利。所以当时德国、日本这些最优秀的公司搞电动车都不顺利，然后当时大家觉得伊隆马斯克作为外行来搞，那可能成吗？这不是坑爹是吧？而且当时大家争论的很多，还是说油电混合才是主流，所以当时对电动车的分析本身被看空。

其次，特斯拉当时刚刚上市，华尔街做空的人非常多啊。原因很简单，当时大家普遍就是把耶隆马斯克这个人看作是个忽悠为主的人，就觉得这家公司只有电动车概念，没有实质的东西，所以华尔街很多人做空。其实这也是我觉得做投资的真谛，因为，做投资就是会反复去验证你对很多事物的看法。甚至像在特斯拉这种风标性的公司，它是非常能验证一个人的三观的。所以更反映了我们当时讲的，投资和财富自由是由三观决定的，现实是最好的老师。比如说如果你当初是不看好电动车，看好油电混合的，不看好马斯克的人，现实本身会给你上一课。然后如果我觉得你真是做空，钱不是太多的情况下，这个学费交的也是值的，因为它很能教会你非常多的东西。

其三，当然还有一个问题，就当时特斯拉并没有公布自己解决充电的方案。当时华尔街很多人怀疑特斯拉能不能解决充电问题。其实当时我们去特斯拉的时候，他们讲的一个最简单的问题，就是当时大家说汽车开起来是有基础设施的，比如说加油站，但是特斯拉你开起来充电站在哪建多少，就全美国建多少个充电站才能满足所谓的特斯拉出行的这个要求呢？所以当时这个基础设施大家也认为是有问题的。这个时候当时对特斯拉这个优秀的公司，以及伊隆马斯克这个优秀的创始人来说，都是一个非常好的买入机会。

比特币的买入机会是 2013 年初，其实当时更早的一些时间当然都是可以的，但是 2013 年初应该是最后一次优质的买入机会了。对于特斯拉和唯品会来说，它还是公司，但是对于比特币来说，它是一个新鲜事物。新鲜事物我们觉得最大的投资机会就是你在判断这个事物本身是优秀的时候，你需要在绝大多数还没有了解到它的时候进入是最重要的。

所以从上面的投资机会来看，投资最好时机，我们讲投资最好的时候往往是好公司，但却没有被绝大多数人看好的时候。

但是你说那这时候大家会问个问题，那绝大多数人不看好，那说不定就是不好的公司，是吧？这时候你就要分析了，我总结出来一个道理，就是你要看对方不看好的原因。就是一些量化问题都是能通过想办法解决的，而不是公司的逻辑内核、概念、发展趋势这些质的东西发生了变化。

就比如说，这个公司的基本的运营逻辑，以及他提出的这个概念。比如说当时唯品会，你说这家公司它提出的 b to c 这种闪购概念能不能成功？这显然是可以成功的。当时唯品会 12 年做的时候，陈欧还准备回来做这个。你要想想，就是陈欧才刚起步的时候，唯品会已经做得很不错了，而且甚至当时包括京东和唯品会算做的比较困难。那当时阿里都准备做天猫，而且天猫做的也都还不错了。所以它的概念本身、逻辑内核、发展趋势都没有问题，其实都没有问题。电商这个 2012 年是一个我们讲的其实也没有那么寒冷的小暖冬，它冻死很多企业。但是很显然是小暖冬，到 2014 年这些公司都开花结果了，包括唯品会涨得非常好，京东也在 2014 年上市了，聚美优品也在 2014 年上市了，阿里巴巴也在 2014 年上市了，所以其实两年之后就很快就好了。

逻辑内核概念、发展趋势，其实公司本身都没有出任何问题，所以这个时候往往就是投资的一个最好时机。
